# hello: welcoming greeter

# EXAMPLE

```
$ bin/hello
Hello, World!
```

# REQUIREMENTS

* [Raku](https://raku.org/) 2025.06.1+
* a UNIX-like environment, e.g. [WSL](https://learn.microsoft.com/en-us/windows/wsl/)
* Install additional development components via `./install`

# TEST

```console
$ ./lair [default]
```

# BUILD

```console
$ ./lair build
```

# CLEAN

```console
$ ./lair clean
```
